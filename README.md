# Flutter Todo App

A simple Todo application built with **Flutter**, demonstrating clean architecture and Firebase integration.

## Features

* Firebase Authentication (Email/Password)
* Firebase Realtime Database
* Riverpod State Management
* Clean Architecture structure
* Add, update, complete, and delete tasks

## Tech Stack

* Flutter
* Firebase Authentication
* Firebase Realtime Database
* Riverpod

## Project Structure

```
lib/
 ├── application
 │    └── providers
 ├── domain
 │    └── models
 ├── infrastructure
 │    └── services
 ├── presentation
 │    ├── screens
 │    └── widgets
 └── core
      └── constants
```

## Getting Started

1. Clone the repository
2. Run:

```
flutter pub get
flutter run
```

3. Make sure Firebase is configured for the project.
